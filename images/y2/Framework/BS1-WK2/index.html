Grids

To use grids we need to:

  1. Create a row

    2. This is attached to a div. It's contents will flex

  2. Put some columns inside our row

    3. Our columns must add up to 12 since BS grid is 12 columns wide

  3. Make different values for our grid

    4. if we want to change layouts on our grids depending on our gridports, we can do this by referencing the correct classes
